[博客](https://www.quxionglie.com)源码文件。 
使用 Hexo + next 主题模板。

使用前调用：
```
npm install
```

运行服务：
```
hexo server
```

浏览器打开[http://localhost:4000/](http://localhost:4000/)查看效果。

发布新文章
```
hexo new [layout] <title>
```
如 hexo new post "talk about chatgpt"


发布
```shell
hexo clean && hexo generate --deploy
```
